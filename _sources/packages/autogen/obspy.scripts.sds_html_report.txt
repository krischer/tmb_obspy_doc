obspy.scripts.sds_html_report
=============================

.. note::
    This script automatically installs during the setup procedure with the
    name ``$ obspy-sds_html_report``. For more info on the command line options,
    please run ``$ obspy-sds_html_report --help``.
    Alternatively you can also execute ``$ python -m obspy.scripts.sds_html_report``.
