obspy.io.xseed.scripts.xseed2dataless
=====================================

.. note::
    This script automatically installs during the setup procedure with the
    name ``$ obspy-xseed2dataless``. For more info on the command line options,
    please run ``$ obspy-xseed2dataless --help``.
    Alternatively you can also execute ``$ python -m obspy.io.xseed.scripts.xseed2dataless``.
