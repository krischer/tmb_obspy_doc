obspy.scripts.print
===================

.. note::
    This script automatically installs during the setup procedure with the
    name ``$ obspy-print``. For more info on the command line options,
    please run ``$ obspy-print --help``.
    Alternatively you can also execute ``$ python -m obspy.scripts.print``.
