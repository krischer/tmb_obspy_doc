obspy.io.xseed.scripts.dataless2xseed
=====================================

.. note::
    This script automatically installs during the setup procedure with the
    name ``$ obspy-dataless2xseed``. For more info on the command line options,
    please run ``$ obspy-dataless2xseed --help``.
    Alternatively you can also execute ``$ python -m obspy.io.xseed.scripts.dataless2xseed``.
